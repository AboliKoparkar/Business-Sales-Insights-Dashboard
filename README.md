# 📊 Business Sales Insights Dashboard (Power BI)

## 🔍 Project Overview

This Power BI dashboard analyzes business sales data to generate meaningful insights and support decision-making.

## 📊 Key Features

* Total Sales, Profit, Quantity, Orders (KPI Cards)
* Sales by Category
* Profit Analysis
* Sales Trend Over Time
* Geographical Sales Distribution

## 🛠 Tools Used

* Power BI
* DAX
* Power Query
